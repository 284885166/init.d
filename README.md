## Only work for Ubuntu Server 10.04

There is some quick install scripts for Ruby on Rails app deploy.

## Usage

Install packages first

```bash
$ ./install_packages
```

### Install MongoDB

```bash
$ ./install_mongodb
```

### Install Redis

```bash
$ ./install_redis
```

### Install Node.JS

```bash
$ ./install_nodejs
```

### Install Ruby by RVM

```bash
$ ./install_rvm
```
