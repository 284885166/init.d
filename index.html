Batch scripts for Ruby production environment install on Ubuntu Server.

Requirements

  • Ubuntu Server 10.04

Usage

Install packages first

$ ./install_packages

Install MongoDB

$ ./install_mongodb

Install Redis

$ ./install_redis

Install Node.JS

$ ./install_nodejs

Install Ruby by RVM

$ ./install_rvm
